# 自媒体文案生成器

一键生成多平台爆款文案，适配小红书、知乎、公众号风格，自动优化标题和标签。

## 核心功能

### 1. 多平台文案生成
- **小红书**：种草文案 + emoji + 话题标签
- **知乎**：专业回答 + 结构清晰 + 权威感
- **公众号**：深度长文 + 排版优化 + 引导关注

### 2. 智能标题优化
- 自动生成 5 个备选标题
- 按平台风格优化（小红书用数字+情绪词，知乎用疑问+利益点）
- 点击率预估评分

### 3. 标签/话题推荐
- 自动提取关键词生成标签
- 热门话题匹配
- 平台专属标签库

### 4. 排版美化
- 自动分段、加 emoji
- 重点内容高亮
- 阅读节奏优化

## 使用方法

### 生成小红书文案

```
帮我写一篇小红书文案，主题是：新手化妆技巧
```

输出包含：
- 5 个爆款标题备选
- 正文（带 emoji）
- 热门话题标签（10-15 个）
- 封面图建议

### 生成知乎回答

```
生成一个知乎回答，问题是：如何高效学习 Python？
```

输出包含：
- 回答结构框架
- 正文内容（专业、有深度）
- 相关话题标签
- 引导互动的话术

### 生成公众号推文

```
写一篇公众号文章，主题是：2024年必读的10本书
```

输出包含：
- 文章标题（3 个备选）
- 文章大纲
- 正文内容
- 排版建议
- 引导关注/分享话术

## 平台风格指南

### 小红书风格
- 开头：痛点/好奇心钩子
- 正文：分点列出，每点配 emoji
- 结尾：互动引导（"姐妹们觉得呢？"）
- 标签：#话题 #关键词（10-15 个）

### 知乎风格
- 开头：直接回答核心问题
- 正文：结构化论述，有数据/案例支撑
- 结尾：总结 + 延伸思考
- 标签：选择精准话题

### 公众号风格
- 开头：故事引入或金句开场
- 正文：深度分析，层次分明
- 结尾：行动号召 + 关注引导
- 排版：小标题、引用、重点标注

## 配置文件

在 `config.json` 中配置：
- `default_platform`: 默认平台（xiaohongshu/zhihu/wechat）
- `tone`: 语气风格（professional/casual/humorous）
- `emoji_style`: emoji 使用频率（high/medium/low）
- `template_folder`: 自定义模板路径
